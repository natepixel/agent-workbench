# e2e/

This folder is for end-to-end tests, such as Playwright.

For the canonical testing ladder used by this template — smoke, unit, integration, and end-to-end — see `tests/README.md`.

## Suggested use
- major user flows
- login/auth flow
- file upload flow
- export flow
- critical regression coverage

## Template reminder
Not every repo needs full e2e coverage immediately, but the folder exists to make the expectation visible from the start.
